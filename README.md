# Hoop-IQ
Quiz where the user can select a decade from the NBA; 2000's, 2010's, 2020's and then test their Hoop IQ from the decade.
# Features
Choose from these 3 NBA Decades
- 2000s
- 2010s
- 2020s
- Each quiz contains 5 multiple-choice questions
- Final score is present once quiz is complete
- Optional answer sheet if desired
# How to Run Code 
1. Make sure Python is installed (version 3.x)
2. Download or clone the repository available
3. Run the program
4. Follow the instructions displayed to select a decade and begin.
